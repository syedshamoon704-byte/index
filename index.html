Phone pe app install karein

Yeh tracker ek HTML file hai — ise apne phone pe app ki tarah install kar sakte ho.

Android (Chrome):
1. Chrome mein yeh file open karein
2. Menu (3 dots) → "Add to Home screen"
3. Done! App icon aa jaega
iPhone (Safari):
1. Safari mein yeh file open karein
2. Share button → "Add to Home Screen"
3. Done! App icon aa jaega
Data baare mein:
Sab data phone ke browser mein save rehta hai (localStorage). Jab tak browser data clear na karein, data safe hai.
